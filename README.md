# Projeto

Java project for the Avanade Spring Boot course. Created as a simple Person API using:

* Spring Boot
* Spring data JPA
* Lombok
* Spring Web
* Mapstruct
* Heroku (deploy)

## Execução

Para executar o código digite no terminal:

```bash
    mvn spring-boot:run
```

É possível acessar a API no link:  
http://localhost:8080/api/v1/person
